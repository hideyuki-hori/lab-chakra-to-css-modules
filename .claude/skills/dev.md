# dev

Next.js開発サーバーを起動し、アプリケーションを確認します。

## 使用方法

```
/dev
```

## 実行内容

### 1. 開発サーバーの起動

```bash
npm run dev
```

デフォルトでは`http://localhost:3000`でアプリケーションが起動します。

### 2. 確認事項

開発サーバー起動後、以下を確認します：

- ✅ Chakra UIが正しく読み込まれているか
- ✅ ページが正しく表示されるか
- ✅ コンソールエラーがないか
- ✅ ホットリロードが動作するか
- ✅ TypeScriptエラーがないか

### 3. 便利なコマンド

```bash
# 型チェック
npm run build

# リントチェック
npm run lint
```

## トラブルシューティング

### Chakra UIが読み込まれない
- `app/layout.tsx`にChakraProviderが設定されているか確認
- `@chakra-ui/react`がインストールされているか確認

### ページが表示されない
- ファイル名が`page.tsx`になっているか確認
- App Router構造になっているか確認

### TypeScriptエラー
- `npm run build`でビルドエラーを確認
- 型定義が正しいか確認

## 注意事項

- ポート3000が既に使用されている場合は、別のポートが使用されます
- 初回起動時は依存関係のインストールに時間がかかる場合があります
