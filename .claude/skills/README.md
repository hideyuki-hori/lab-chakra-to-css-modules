# Skills (Approach A: Direct Migration)

CSS Modules直接移行用のスキル一覧

## 利用可能なスキル

| スキル | 説明 |
|--------|------|
| `/migrate` | ページ移行の全体フロー |
| `/screenshot` | スクリーンショット撮影 |
| `/share` | プロジェクト概要をクリップボードに出力 |

## よくある作業フロー

### 新しいページの移行を開始

```
/migrate profile 3
```

### スクリーンショットだけ撮影

```
/screenshot before 3 profile
/screenshot after 3 profile
```
